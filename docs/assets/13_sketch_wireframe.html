합의 사항

스케치 토론에서 팀이 맞춘 내용 (초안 — 워크숍에서 확정)

디자인 방향

  • 감성·아날로그 무드의 따뜻한 인터랙티브 UI
  • 이미지가 주인공 — 텍스트·크롬은 최소화
  • 완성까지의 기대감(진행률·실시간 미리보기)을 연출

공통 UI 규칙

  • 모바일 세로 우선, 단일 컬럼 흐름
  • 주요 행동은 하단 고정 CTA 1개로 명확히
  • 이미지는 카드/썸네일 그리드, 라운드 8px 통일
  • 인터랙션 요소는 accent 색으로만 구분

구현 시 유의

  • 테스트는 비로그인 진행 → 저장 시점 게이팅
  • 생성 지연 대비 임시저장·진행 상태 필수
  • 공유 링크 썸네일(OG) 미리 설계
  • 히스토리 삭제는 되돌리기 확인 절차
