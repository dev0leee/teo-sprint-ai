## 서비스 목적

> 내가 생각하는 우리 서비스의 궁극적인 목적은 무엇인가요?

- (claude) 질문과 선택만으로 "나의 무드(취향)"를 시각적인 추구미 보드로 손쉽게 만들어, 핀터레스트에서 몇 시간씩 이미지를 찾던 진입장벽을 없애는 것.
- (claude) "내가 생각하는 나"와 "친구들이 생각하는 나"를 하나의 보드로 비교하며, 나를 발견하고 서로를 새롭게 알아가는 경험을 주는 것.
- (claude) 혼자 써도 재미있는 도구이자, 친구와 함께 쓰면 더 즐거운 놀이가 되어 주변 사람들과 나를 공유·소통하게 하는 것.
- (녕녕) 나를 발견하고 서로를 더 이해하는 경험 = **놀이도구**. "내가 생각하는 나"와 "친구가 생각하는 나"를 하나의 무드보드로 시각화·비교해 자기 이해와 서로에 대한 이해를 돕고, 취향을 정리하는 도구이자 함께 즐기는 놀이로서 나를 표현·공유하는 새로운 방식을 제공.
- (류하) 나 자신을 언어가 아닌 **이미지로 발견하고 표현**하게 하는 것.
- (몌굥) 자신의 추구미가 뭔지 **아직 잘 모르는 사람**이, 질문과 친구의 시선을 통해 "내가 어떤 사람인지"를 발견하고 하나의 무드보드로 표현하게 하는 서비스. (키워드: 놀이 도구 / 이미지 / 추구미를 발견해가는)

## 서비스 대상

> 내가 생각하는 우리가 만들 서비스의 주요대상은 누구인가요?

- (claude) 비전보드·추구미 보드 유행에 관심은 있지만, 직접 만들기엔 시간·감각이 부담스러워 시작하지 못한 2030 SNS 유저.
- (claude) 자기 취향·정체성을 시각적으로 표현하고 공유하는 걸 좋아하는 사람들(핀터레스트·감성 콘텐츠 소비층).
- (claude) 친구·커플·동아리·MT처럼 서로를 새롭게 알아가는 놀이를 찾는 소규모 그룹.
- (녕녕) **나를 알고 싶고 친구를 더 알고 싶은 사람들.**
  - Primary: 10~30대 여성 / MBTI·가치관 테스트·추구미·비전보드 같은 콘텐츠를 좋아하는 사람 / SNS에 자신의 취향을 공유하는 걸 좋아하는 사람.
  - Secondary: 친구들 / 커플 / 동아리 / MT·워크숍 참가자 / 생일 선물이나 추억을 특별하게 남기고 싶은 사람.
- (류하) **자기표현 트렌드에 익숙한 MZ세대** / 친구에게 애정이 있는 사람.
- (몌굥) 확산 단계별 타겟(공유 기반).
  - 1차: 내 추구미가 뭔지 잘 모르는 '발견형' + 생일·MT·워크숍처럼 함께 놀 계기가 있는 대학생 그룹·친한 친구 무리·커플 등 '**관계·놀이형**'.
  - 2차: 친구·모임 단위로 확산력이 있는 사람.
  - 3차: 이미 추구미가 확실하고 자신을 드러내고 싶은 '전시형'.
  - → 관계·놀이형을 1차 타겟으로 하며, 전시형·발견형까지 자연스럽게 흡수.

## 서비스 핵심 가치

> 내가 생각하는 우리가 만들 서비스가 추구해야 할 핵심가치는?

- (claude) **결국 이뻐야 한다** — 감성적이고 "느좋"한 결과물의 완성도가 서비스의 첫인상이자 공유를 부르는 핵심.
- (claude) **쉬운 진입, 낮은 이탈** — 공유된 보드를 보고 나도 써보고 싶게, 진입 후엔 복잡한 플로우 없이 바로 핵심 기능(내 보드 만들기)까지 도달.
- (claude) **혼자도·함께도 재미** — "내가 만드는 나"와 "친구가 만들어주는 나"의 비교라는 킥으로, 만드는 과정 자체가 즐거운 경험이 되게.
- (녕녕) **자기 발견**(질문으로 스스로의 가치관·취향을 알아가는 경험) / **표현**(텍스트가 아닌 이미지와 무드로 자신을 표현) / **공감**(친구가 보는 나와 내가 보는 나를 비교하며 서로를 이해) / **재미**(단순 테스트가 아니라 게임처럼 즐기는 가치여정) / **공유**(결과를 친구들과 쉽게 공유하고 함께 즐길 수 있음).
- (류하) **자기 발견의 진정성**(내가 직접 나에 대해 답하니 더 솔직하게) / **결과물의 자기증명력**(텍스트 소개보다 훨씬 설득력 있게 보여주기) / **시간에 따라 달라지는 존재**("이번 계절의 나 무드보드"처럼 시즌·주기 컨셉).
- (몌굥) ① 나를 무드로 손쉽게 드러낸다 ② 과정에서 내가 어떤 사람인지 알게 된다 ③ 친구와 공유를 통해 서로를 새롭게 알아간다 ④ 혼자도 재밌고 공유하면 더 재밌다.

> **논의 메모**
> - (몌굥) 원동력(발견)과 목적(놀이)이 뭔가 한 방향이 아닌 것 같다.
> - (류하) 발견의 대상이 나인지 상대인지에 따라 질문의 방향이 달라져야 함.
> - (몌굥) 검색해봤을 때 겹치는 서비스는 없었음. 이유가 핀터레스트가 있어서인데, 핀터레스트 수동작업은 어렵기 때문에 우리 서비스가 '쉽게 만들기'라는 강점을 가져갈 수 있음.
