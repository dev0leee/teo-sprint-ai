# Step 05. 기존 고객 여정

> 목적: `docs/04`에서 정의한 서비스 목적을, **지금** 사용자가 어떤 방식으로 달성하는지 정리한다.

> **Guide:** **우리가 만들 서비스의 여정이 아니다.** (우리 서비스 여정은 Step 12 `docs/12_customer_journey_map.md`) 유사 서비스·대체제·우회 방식마다 **여정을 하나씩** 두고, 단계별 사용자 행동·생각/감정을 표로 정리한다. 각 단계는 상세하게 작성한다. 이 내용을 토대로 Step 06에서 Pain/Wow 인사이트를 추출한다.

# 클로드

## 여정 1

> **핀터레스트로 추구미/비전보드 직접 만들기** — "나를 이미지로 표현·정리"를 지금 가장 많이 쓰는 방식

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 계기·영감 | SNS·릴스에서 남의 추구미 보드를 보고 "나도 만들어보고 싶다"고 생각 | 예쁘다, 나도 저런 무드로 정리하고 싶다 (기대·설렘) |
| 검색 | 핀터레스트에서 "추구미", "vision board", 무드 키워드로 이미지를 검색 | 뭐라고 검색해야 내 느낌이 나오지? 키워드가 애매하다 (막막함) |
| 이미지 수집 | 마음에 드는 핀을 하나씩 저장, 보드(폴더)에 모음 | 이미지는 많은데 내 취향인지 확신이 안 선다 (선택 피로) |
| 배치·정리 | 별도 앱(캔바·포토샵·모바일 편집기)으로 이미지를 콜라주처럼 배치 | 예쁘게 배치하는 게 제일 어렵고 시간이 오래 걸린다 (지침) |
| 완성·저장 | 완성본을 캡처/이미지로 저장 | 몇 시간 썼는데 이게 정말 '나'를 표현한 건가? (애매한 성취감) |
| 공유 | 인스타 스토리·프로필 등에 올려 자랑 | 반응은 좋지만 다시 만들라면 엄두가 안 난다 (진입장벽) |

## 여정 2

> **MBTI·심리/성격 테스트로 나 알아보기** — "질문을 통한 자기 발견"을 지금 대체하는 방식

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 발견 | 친구가 공유한 테스트 링크(퍼스널컬러·MBTI·심리테스트 등)를 SNS에서 접함 | 재밌겠다, 나도 해봐야지 (가벼운 호기심) |
| 응답 | 질문 몇 개에 직관적으로 응답 | 질문이 쉽고 빠르게 끝나서 좋다 (몰입) |
| 결과 확인 | 유형·키워드·설명 결과 화면을 확인 | 오 은근 맞네 / 근데 다 비슷비슷한 결과 같기도 (공감 반, 아쉬움 반) |
| 저장·공유 | 결과 이미지를 캡처해 스토리에 공유 | 친구들도 해봤으면, 결과 비교하고 싶다 (공유 욕구) |
| 친구와 비교 | 친구 결과와 서로 캡처를 비교 | 너는 이거 나왔네~ 하고 잠깐 얘기하고 끝 (일회성·휘발) |
| 이탈 | 며칠 뒤엔 결과를 잊음, 남는 결과물이 없음 | 재밌었지만 '나'로 남는 건 없다 (아쉬움) |

## 여정 3

> **롤링페이퍼·익명질문으로 "친구가 보는 나" 모으기** — 생일·모임에서 쓰는 방식

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 계기 | 생일·MT·송별 등에서 "친구들이 보는 나"를 모아보고 싶어짐 | 애들은 나를 어떻게 볼까? (궁금·설렘) |
| 도구 준비 | 종이 롤링페이퍼, 구글폼, 인스타 익명질문 스티커 등을 준비 | 매번 도구가 제각각이고 세팅이 번거롭다 (번거로움) |
| 요청 | 친구들에게 "나에 대해 한마디/이미지 남겨줘" 부탁 | 부탁하기 살짝 민망하고, 참여율이 걱정된다 (부담) |
| 수집 | 친구들이 텍스트 위주로 답을 남김 | 글은 오는데 '무드'나 이미지로는 안 모인다 (한계) |
| 취합 | 흩어진 답을 직접 읽고 정리 | 취합·정리를 다 수작업으로 해야 한다 (수고) |
| 간직·공유 | 캡처하거나 종이째 보관 | 따뜻하지만 다시 보기 어렵고, 내 보드랑 비교할 방법이 없다 (휘발·단절) |

# 녕녕

## 여정 1

> **친구가 만들어줌** — 친구의 부탁을 받아 "친구를 위한 추구미 보드"를 직접 만들어주는 여정

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 요청 받음 | 친구가 "내 추구미 보드 만들어줘"라고 부탁 | 이게 뭐야...? (생소함) |
| 검색·파악 | 추구미 보드가 뭔지 검색해서 알아봄 | 아하, 이게 추구미 보드구나 (이해) |
| 키워드 설정 | 사진을 어떻게 다 고르나 막막 → 우선 보드마다 키워드부터 설정 | 사진 언제 다 고르지..? 일단 키워드부터 (막막·정리 시도) |
| 이미지 찾기 | 핀터레스트·구글·네이버에서 키워드당 사진 3개 정도 고름 | 검색처가 제각각, 고를 게 너무 많다 (수고) |
| 후보 선별 | 후보군 중 가장 어울리는 것 고르기 → 완성 | 완성!!! 근데 엄청 오래 걸렸다 ㅠ (성취+소진) |
| 조합 도구 탐색 | 사진 조합 어플이 뭐가 좋은지 알아봐야 함 → 설치 | 이것도 평소 안 해봐서 뭘 써야 할지 모르겠다 (인지부채) |
| 조합 | 다운받은 사진들을 어플에서 조합 | 조합까지 해야 끝이구나 (번거로움) |
| 공유 | 완성본을 친구에게 공유 | 친구가 웃음 ㅋㅋ 좋아함 (뿌듯·보람) |
| 소진 | "하 불태웠다.. 해주는 건 너가 끝이다" → 친구가 SNS에 공유 | 보람은 있지만 다신 못 하겠다 (완전 소진) |

> **녕녕 발표 정리:** AI로 이미지를 생성하는 방법도 있을 듯.

# 몌굥

## 여정 1 — 내 추구미 보드를 만들고 싶음

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 핀터레스트 검색 | 추구미 보드를 만들려고 핀터레스트에서 검색 | 어떤 키워드로 검색해야 원하는 이미지가 나오는지 모르겠다 (막막함) |
| 이미지 수집 | 보드에 이미지를 쌓음 | 쌓고 나니 한눈에 확인하기 어렵다 (정리 안 됨) |
| 인스타 탐색 | 인스타에서 이미지를 찾아봄 | 평소엔 예쁜 게 잘 보이는데 막상 필요할 때 못 찾겠다 (아이러니) |
| 수제 제작 | 이미지를 직접 오리고 붙여 수제로 만듦 | 피곤하다(체력 이슈), 보관이 어렵다 / 그래도 만드는 과정에서 원하는 추구미 윤곽이 정리되는 느낌은 있다 (양가감정) |
| 대체 플랫폼 | 다른 취향찾기 플랫폼을 이용 | 딱 내가 원하는 느낌이 아니다 (아쉬움) |

## 여정 2 — 친구의 추구미가 알고 싶음

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 직접 질문 | 친구에게 직접 물어봄 | 답변을 회피 당함 (무안·막힘) |
| 플랫폼 이용 | 취향 찾기 플랫폼을 이용 | 단편적인 결과물만 받는다 (부족함) |

## 여정 3 — 변해가는 추구미를 알고 싶음

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 수동 기록 | 변해가는 추구미를 수동으로 기록 | 기록해두면 흐름을 볼 수 있을 텐데 (기대) |
| 지속 실패 | 꾸준히 기록하지 못함 | 안 하게 돼서 결국 모르겠다 (자책) |
| 기록 유실 | 남긴 기록마저 유실됨 | 남는 게 없다 (허무) |

> **몌굥 요약:** ① 그냥 이미지 찾는 게 힘듦 ② 보관도 힘듦 ③ 꾸준히도 힘듦 ④ 그게 이쁘지가 않음(추구미 트렌드).
> 참고 — 핀터레스트에서 **outfit / food / nails / quote** 네 키워드를 검색해 이미지를 추구미로 삼음.
>
> **몌굥 발표 정리:** 손쉽게 만드는 게 좋은데 인지부채가 쌓임. / 어렵게 만드는 데서 나오는 뿌듯함·소중함·한땀한땀. / 이미지 찾기 힘듦 → 이미지는 찾기 어렵게 만들어야 하나..? / 전체적인 이미지 fit이 맞아야 함 / 키워드가 중요.. / 사진 광, 느낌.
